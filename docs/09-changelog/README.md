# 09-changelog 变更日志

## 用途
按版本追踪所有变更，便于回溯

## 文件命名
`v{版本号}.md`

## 版本号规则
`v{主版本}.{次版本}.{修订版}.md`

- `v1.0.0.md` - 初始版本
- `v1.1.0.md` - 新增功能
- `v1.2.1.md` - Bug 修复

## 文件模板
```markdown
# v{版本} - {发布日期}

## 新增
- {功能 1}
- {功能 2}

## 改进
- {改进 1}
- {改进 2}

## 修复
- {Bug 1}
- {Bug 2}

## 变更
- {变更 1}
- {变更 2}

## 移除
- {移除 1}

## 贡献者
- {贡献者 1}
- {贡献者 2}

## 关联 PR
- #{PR 号} - {PR 标题}
```

## 示例
```
09-changelog/
├── v1.0.0.md
├── v1.1.0.md
├── v1.2.0.md
└── v1.2.1.md
```
